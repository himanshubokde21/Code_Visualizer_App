About This App

CodeNeura lets you import a project folder, explore its structure visually, view source files with syntax highlighting, generate a basic flowchart, and see a mock AI analysis.

Features

  • Drag / select a folder to visualize its hierarchy.
  • Syntax‑highlighted code viewer powered by Prism.js.
  • Auto-generated flowchart placeholder using Mermaid.js.
  • Resizable panels for a customizable layout.
  • Light and dark theme support.
